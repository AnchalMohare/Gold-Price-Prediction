# Gold Price Prediction using Machine Learning

## Project Overview

This project predicts gold prices using Machine Learning techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Algorithm

* Random Forest Regressor

## Features

* Data Visualization
* Correlation Heatmap
* Gold Price Trend Analysis
* Prediction Model
* Actual vs Predicted Comparison

## Dataset

Gold price dataset containing:

* SPX
* GLD
* USO
* SLV
* EUR/USD

## Results

The model achieved high accuracy using Random Forest Regression.
